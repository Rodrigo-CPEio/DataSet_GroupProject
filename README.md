## 📋 Sobre o Projeto
Este projeto foca-se na análise exploratória e na aplicação de modelos de Machine Learning utilizando Python.
O objetivo principal é identificar variáveis-chave, adaptar modelos estatísticos e realizar uma análise crítica dos resultados obtidos.

O trabalho está dividido em três componentes principais:
1. **Análise Estatística e Exploratória (40%)**
2. **Modelagem de Machine Learning (60%)** 
3. **Escrita de Artigo Científico**
